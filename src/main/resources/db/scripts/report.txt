При добавлении 1_000_000 заявок Heap Memory изменяется скачками прыгает до 150mb и опускается до 5-50mb, CPU подпрыгнул до 9% потом упал на 6% а после на 0, Classes Threads особо не изменились
При удалении 1_000_000 заявок Heap Memory подпрыгныл до 600mb упал до 400mb затем опять до 600mb потом на 100mb, CPU до 16% затем до 4% и потом до 0, Classes Threads особо не изменились
При показе всех заявок, графики особо не изменились.
Анализ проводился инструментом jconsole.
